# Contributing

Thanks for your interest! This is a single-page React app (vanilla, no build step).

## Local dev
```bash
python3 -m http.server 8765
# open http://127.0.0.1:8765/Notebook%20Board.html
```

## How to contribute
1. Fork, create a feature branch.
2. Make your change. Test in the browser.
3. Open a PR with a short description.
